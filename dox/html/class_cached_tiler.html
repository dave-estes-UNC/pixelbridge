Detailed Description

This tiler will split provided frames into tiles and update the NDDI display.

It organizes the display into dimensions matching the tile size in the x and y directions and then the cache size in the z direction. It will then update the coefficient plane so that a region of the display maps to the appropriate tile in the cache.
